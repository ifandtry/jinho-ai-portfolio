# Jinho Lee AI Builder Portfolio

Premium portfolio website for AI, automation, operations systems, workflow design, and product thinking work.

## Stack

- Next.js App Router
- React
- TypeScript
- Tailwind CSS
- Framer Motion
- lucide-react
- React Three Fiber

## Pages

- Home
- Project detail pages
- Story
- Contact
